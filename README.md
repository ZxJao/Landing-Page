# 🍽️ Restaurant Landing Page Uma Landing Page moderna e responsiva desenvolvida para apresentar um restaurante de forma elegante e intuitiva. O projeto foi criado com HTML5, CSS3 e JavaScript, com foco na prática de desenvolvimento Front-End e na criação de interfaces dinâmicas e interativas. ## 🚀 Badges ![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black) ![Status](https://img.shields.io/badge/Status-Concluído-success?style=for-the-badge) --- ## 📖 Sobre o Projeto A Restaurant Landing Page foi desenvolvida para simular a página inicial de um restaurante, apresentando informações como o cardápio, destaques dos pratos, depoimentos de clientes e uma navegação intuitiva entre as seções da página. O projeto tem como objetivo praticar a construção de layouts modernos e responsivos utilizando HTML, CSS e JavaScript. --- ## ✨ Funcionalidades - Layout moderno e responsivo; - Navegação interativa entre as seções; - Seção de apresentação do restaurante; - Cardápio detalhado; - Depoimentos de clientes; - Rodapé personalizado; - Interface dinâmica e amigável ao usuário. --- ## 🛠️ Tecnologias Utilizadas - HTML5 - CSS3 - JavaScript --- ## 📁 Estrutura do Projeto ```bash Restaurant-Landing-Page/ │ ├── css/ │ ├── cabeçalho.css │ ├── home.css │ ├── menu.css │ ├── testimonials.css │ ├── rodape.css │ └── estilo.css │ ├── index.html ├── script.js └── README.md ``` --- ## ▶️ Como Executar o Projeto 1. Clone este repositório: ```bash git clone https://github.com/seu-usuario/restaurant-landing-page.git ``` 2. Acesse a pasta do projeto: ```bash cd restaurant-landing-page ``` 3. Abra o arquivo `index.html` em seu navegador. --- ## 🎯 Objetivos do Projeto Este projeto foi desenvolvido para praticar conceitos fundamentais do desenvolvimento Front-End, tais como: - Estruturação de páginas com HTML5; - Organização de estilos em múltiplos arquivos CSS; - Desenvolvimento de interfaces responsivas; - Manipulação de elementos com JavaScript; - Criação de layouts modernos e intuitivos. --- ## 👨‍💻 Autor Desenvolvido por João Pedro.
