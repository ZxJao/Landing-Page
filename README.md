# 🍽️ Landing Page - Restaurante

Projeto de uma **landing page para restaurante** desenvolvido com **HTML, CSS e JavaScript**.

## 🚀 Tecnologias
- HTML
- CSS
- JavaScript

## 📁 Como usar
1. Clone ou baixe o repositório  
2. Abra o arquivo `index.html` no navegador  

## 🎯 Objetivo
Praticar desenvolvimento web criando uma página de restaurante moderna, com interatividade usando JavaScript.

## ✨ Funcionalidades
- Menu de navegação  
- Seção de cardápio  
- Animações simples  
- Interatividade com JavaScript  
- Layout responsivo
