# 🍽️ Restaurant Landing Page

Uma Landing Page moderna e responsiva desenvolvida para apresentar um restaurante de forma elegante e intuitiva. O projeto foi criado com foco na prática de desenvolvimento Front-End, utilizando HTML, CSS e JavaScript para construir uma interface dinâmica e agradável ao usuário.

## 📖 Sobre o Projeto

A Restaurant Landing Page foi desenvolvida para simular a página inicial de um restaurante, apresentando informações como cardápio, destaques do estabelecimento e opções de navegação intuitiva. O projeto tem como principal objetivo praticar a criação de interfaces modernas, responsivas e interativas utilizando tecnologias fundamentais do desenvolvimento Front-End.

---

## ✨ Funcionalidades

- Layout moderno e responsivo;
- Seção detalhada do cardápio;
- Navegação dinâmica entre as seções da página;
- Design intuitivo e agradável ao usuário;
- Interface otimizada para diferentes tamanhos de tela;
- Elementos interativos desenvolvidos com JavaScript.

---

## 🛠️ Tecnologias Utilizadas

- HTML5

- JavaScript
