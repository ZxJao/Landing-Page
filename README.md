# 🍔 Restaurant Landing Page

Uma Landing Page responsiva desenvolvida para um restaurante, criada com HTML5, CSS3 e JavaScript. O projeto apresenta um design moderno e intuitivo, destacando o cardápio, depoimentos de clientes e informações do estabelecimento, proporcionando uma experiência agradável ao usuário.

## 📖 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais do desenvolvimento Front-End, como a construção de layouts responsivos, organização de estilos em múltiplos arquivos CSS e manipulação de elementos utilizando JavaScript.

A Landing Page foi projetada para apresentar um restaurante de maneira moderna e profissional, facilitando a navegação entre as seções do site.

---

## ✨ Funcionalidades

- Layout moderno e responsivo;
- Navegação fluida entre as seções da página;
- Seção inicial de apresentação do restaurante;
- Cardápio organizado e visualmente atrativo;
- Depoimentos de clientes;
- Rodapé personalizado com informações adicionais;
- Interface dinâmica desenvolvida com JavaScript.

## ▶️ Como Executar o Projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/Landing-Page.git
```

2. Acesse a pasta do projeto:

```bash
cd Landing-Page
```

3. Abra o arquivo `index.html` em seu navegador.
